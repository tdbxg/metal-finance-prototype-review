# 金属加工订单财务小程序原型

这是网页原型的微信小程序版本，适合先用微信开发者工具预览。

## 打开方式

方式一：导入整个仓库或原型目录。

1. 打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择仓库根目录，开发者工具会根据根目录的 `project.config.json` 找到 `miniprogram/`。

方式二：只导入小程序目录。

1. 打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择本文件夹：`miniprogram`。
4. AppID 可先使用测试号或游客模式，正式上线前再替换为企业自己的小程序 AppID。

## 官方结构对照

根据微信官方「目录结构」文档，小程序由根目录的 `app` 和多个 `page` 组成：

- `app.js`：小程序逻辑，必需。
- `app.json`：小程序公共配置，必需。
- `app.wxss`：小程序公共样式，非必需，当前已配置。
- 每个页面必须保持同路径同文件名，例如 `pages/quote/index.js`、`pages/quote/index.wxml`、`pages/quote/index.json`、`pages/quote/index.wxss`。

当前目录已经按这个规则配置。

## 当前页面

- 总览：收入、成本、毛利、未收款和订单利润。
- 报价：客户选择、材料选择、多零件报价、历史报价保存与套用。
- 客户：客户档案、客户 Excel/CSV 文件选择入口。
- 材料：材料库存、材料价格表导入入口。
- 拍照单据：生产单、收货单、发票、银行回单的拍照上传与待复核队列。

## 当前数据存储

当前为原型阶段，数据存在微信小程序本地缓存 `wx.setStorageSync`。
后续正式使用建议接入云开发数据库或自有后端，至少拆成客户、材料、订单、报价、单据、发票、收付款几张表。
